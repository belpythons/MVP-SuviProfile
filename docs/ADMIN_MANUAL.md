## ğŸ“š Mengelola Kursus

### Melihat Daftar Kursus

1. Dari menu samping, klik **"Kursus"**
2. Anda akan melihat tabel berisi semua kursus

### Mengedit Kursus

1. Klik tombol **"Edit"** pada kursus yang ingin diubah
2. Anda dapat mengubah:
   - **Judul Kursus**: Nama program pelatihan
   - **Deskripsi**: Penjelasan tentang kursus
   - **Harga**: Biaya program (dalam Rupiah)
   - **Status Aktif**: Toggle untuk menampilkan/menyembunyikan di website

### Mengupload Gambar Kursus

1. Di halaman edit kursus, cari bagian **"Gambar Kursus"**
2. Klik tombol **"Upload Gambar"** atau **"Ganti Gambar"**
3. Pilih file gambar dari komputer Anda
   - Format: PNG, JPG, atau WEBP
   - Ukuran maksimal: 4MB
4. Tunggu hingga upload selesai
5. Gambar akan muncul sebagai preview
6. Jangan lupa klik **"Simpan Perubahan"** setelah selesai

### Mengatur Syllabus (Materi Kursus)

Syllabus adalah daftar modul/topik yang akan dipelajari peserta.

1. Di halaman edit kursus, scroll ke bagian **"Materi Syllabus"**
2. **Menambah modul baru**: Klik tombol **"Tambah"**
3. **Mengisi nama modul**: Ketik nama topik di kolom input
4. **Menghapus modul**: Klik ikon ğŸ—‘ï¸ (tempat sampah) di sebelah kanan
5. **Urutan modul**: Modul ditampilkan sesuai urutan di form
6. Klik **"Simpan Perubahan"** setelah selesai

**Contoh syllabus:**
1. Pengenalan Microsoft Excel
2. Rumus dan Fungsi Dasar
3. Pivot Table dan Chart
4. Studi Kasus dan Praktik

## ğŸ“‹ Melihat Leads (Data Pendaftar)

Leads adalah data calon peserta yang menghubungi via WhatsApp.

### Cara Melihat Leads

1. Dari menu samping, klik **"Leads"**
2. Anda akan melihat tabel dengan informasi:
   - **Nama**: Nama calon peserta
   - **No. WhatsApp**: Nomor yang bisa dihubungi
   - **Kursus**: Program yang diminati
   - **Tanggal**: Waktu pendaftaran
   - **Status**: NEW, FOLLOWED_UP, CONVERTED, atau CLOSED

### Memahami Status Lead

| Status | Arti |
|--------|------|
| **NEW** | Lead baru, belum dihubungi |
| **FOLLOWED_UP** | Sudah ditindaklanjuti/dihubungi |
| **CONVERTED** | Sudah mendaftar sebagai peserta |
| **CLOSED** | Tidak jadi mendaftar |
